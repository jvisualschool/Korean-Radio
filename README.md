# 📻 K-RADIO: 대한민국 실시간 라디오 플레이어

**K-RADIO**는 최신 웹 기술을 활용하여 제작된 프리미엄 실시간 라디오 스트리밍 웹 애플리케이션입니다. 깔끔하고 모던한 UI를 통해 대한민국의 주요 라디오 방송을 고음질로 청취할 수 있습니다.

---

## ✨ 주요 특징

- **실시간 스트리밍**: EBS, CBS, KBS, MBC(광주), SBS, TBS 등 주요 방송국 채널 완벽 지원.
- **7가지 다이내믹 테마**: 사용자의 취향에 맞춰 변경 가능한 다양한 디자인 시스템 테마 제공.
  - `Deep Navy`, `Classic Radio`, `Mono`, `Dark`, `Line Art`, `Luxury Gold`, `Forest Mint`
- **반응형 컴팩트 UI**: 하단 고정형 컨트롤러와 가로형 채널 선택기로 어느 기기에서나 일관된 사용자 경험 제공.
- **애니메이션 비주얼라이저**: 실시간 오디오 재생에 맞춰 움직이는 다이내믹 이퀄라이저 애니메이션.
- **스플래시 모달**: 앱 정보와 기술 스택을 확인할 수 있는 미려한 스플래시 화면.

## 🛠 사용된 기술 (Tech Stack)

- **Frontend**: [React](https://reactjs.org) (v18+)
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **Build Tool**: [Vite](https://vitejs.dev)
- **Styling**: Vanilla CSS (Custom Variable Design System)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev)

## 🚀 빠른 시작

### 1. 프로젝트 복제 및 설치
```bash
git clone https://github.com/your-repo/RADIO.git
cd RADIO
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```
실행 후 브라우저에서 `http://localhost:5173`으로 접속하세요.

### 3. 프로젝트 빌드
```bash
npm run build
```
